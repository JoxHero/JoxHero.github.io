Markdown初探

  • Markdown

起初我也想学着大牛搭建自己的博客，比如Hexo、简书之类的，但是当博客建立起来的时候（我的博客），却迟迟没有去填充，一方面原因是自己的学识不够、技术不强、经验不足、语言表达不通畅！！！ 但是做为一个中华民族好青年，社会主义接班人，我告诉我自己要学习，学习，学习。学习之后要积累，所以我要去学习新技术，学习中去积累。 好了说了这么多，言归正传，像简书、Hexo、这样的网站博客，编写文章的时候都需要用到Markdown。 所以今天就Markdown做一下简单的了解。所谓的了解也就是看别人的积累，搬过来成为自己的，说的自己挺不要脸的哈。因为我自己是个小白，所以只能这么无耻了。

##Markdown是啥？
Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。。。。。。[请自行百度]

###Markdown简单常用语法

####1. 斜体
如果你想让某些文字变成斜体，只需要用一颗星“”包围着他们就好了。比如hello world！将显示成hello world*！

####2. 粗体
如果你想让某些文字变成粗体，只需要用两颗星“”包围着他们就好了。比如hello world！将显示成hello world**！

####3. 链接
[链接描述] (link url)
比如：[百度] (http://www.baidu.com) 将显示成个人主页
注意 “]”和“(”之间不能有空格

####4. 图片
图片与链接之间的区别是，在前面多了一个”!”
![链接描述] (imageurl)将显示imageurl所链接到的那个图片，如：![美国队长] (http://chriszou.com/images/ca.jpg) 将显示：

####5. 标题
如果你想让文字变得粗大，像文章的标题一样，可以在前面加上“#”，如:
–) # 1分钟学会markdown
将显示成：

#1分钟学会markdown
需要注意的是，#和后面的文字之间需要有一个空格,必须在行首。
前面一个“#”为一级标题，是最粗最大的。二级标题为两个“#”，以此类推，共6级
